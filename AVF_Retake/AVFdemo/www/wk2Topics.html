  • 3. Devices: Please list two devices (and some of their features) that are made specifically to support Accessibility.

    One feature that I take advantage of is the clicking of the fn button twice on the Mac Book Pro Mountain Lion allows you to speak to text. I use the everyday and is nice when someone else types so fast it hard to keep up, you can speak to text and stay up with the conversation. I have RA and a lot of times it is hard to use my fingers by typing for a long period of time. My fingers seem to have dyslexia so I love taking advantage of this function on the MacBook Pro.

    Now on my android phone a feature that can be used is the font size feature to raise the font for people that have visual impairment. You can access this function in the settings in the accessibility. My wife seems to use this function a lot because she doesn't like to wear her classes so makes it easier for her to read.
